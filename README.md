TtAC (Text to Audio Converter)

Самый простейший преобразователь текста в речь с сохранением в MP3/WAV.

## Зависимости

```csharp
using System.Speech;
using NAudio;
using NAudio.Lame;
```

## ✨ Возможности

- ⏯️ Воспроизведение текста в голос / Стоп
- 🎙️ Выбор голоса (Встроенные в Windows)
- 💾 Сохранение в MP3/WAV
